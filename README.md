## vim

* Install vim plug

`
curl -fLo ~/.vim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
`

* Run `:PlugInstall`

## tmux

* Install tpm

`git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`
